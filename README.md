# Acowale CRM - Customer Feedback Platform

A full-stack Customer Feedback Platform developed as part of the Acowale Machine Test.

## 📌 Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router
- Recharts

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- Uvicorn

---

## ✨ Features

- Submit Customer Feedback
- View All Feedback
- Dashboard Analytics
- Category-wise Feedback Count
- Recent Feedback List
- REST API
- Interactive Swagger Documentation

## ⚙️ Backend Setup

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

Backend URL:

```
http://127.0.0.1:8000
```

---

## 💻 Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend URL:

```
http://localhost:5173
```
